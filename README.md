# PazKids Web Profesional

Página web estática para GitHub Pages.

## Estructura

- `index.html`: contenido principal.
- `css/styles.css`: estilos visuales.
- `js/main.js`: menú móvil y FAQ.
- `assets/`: imágenes de marca y fotografías reales.

## Publicar en GitHub Pages

1. Crea un repositorio nuevo, por ejemplo: `pazkids-web`.
2. Sube todos estos archivos.
3. En GitHub entra a `Settings` > `Pages`.
4. En `Build and deployment`, elige `Deploy from a branch`.
5. Selecciona `main` y carpeta `/root`.
6. Guarda. GitHub te dará una URL pública.
